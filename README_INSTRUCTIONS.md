## To open the file, please follow these steps:
1️⃣ Once you have the entire repository downloaded on your device, open the folder and be sure to install pygame and flask via the terminal/cmd to let the program run.  
2️⃣ On your terminal, type **python app.py** or **py app.py** to run the localhost.    
3️⃣ Please find the html file **a.smth.html** and open it. This will redirect you to your localhost and run the wheel.   
4️⃣ Press the spin button for a random game.  
5️⃣ Play the game you got!
